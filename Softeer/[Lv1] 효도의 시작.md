# 문제
어버이날 효도의 시작은 부모님께 사랑의 메세지를 남기는 것입니다.
사랑의 마음을 담아 엄마 아빠 사랑해요!를 출력하는 프로그램을 작성해보세요

# 풀이
Softeer 1단계부터 풀어보려고 한다.
너무 쉬운 문제라 따로 풀이는 적지 않겠다.

```
#include<iostream>

int main(int argc, char** argv)
```
